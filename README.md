- Explorar bibliotecas e frameworks
- Desenvolver boas práticas de programação
- Registrar minha evolução como desenvolvedor

## 🚀 Tecnologias

- Python 3

- Git

Este repositório está em constante atualização à medida que novos conteúdos são estudados e novos desafios são resolvidos. O objetivo é manter um histórico organizado do aprendizado e da prática em Python.
